# ITC-Hotel_Power_BI_Dashboard


## Project Overview
This project is a Power BI Dashboard developed to analyze the business performance of ITC Hotels. The dashboard provides insights into revenue trends, occupancy rates, room category performance, cancellations, and booking behavior.

<img width="1280" height="720" alt="Image_power_bi" src="https://github.com/user-attachments/assets/6d5726ec-c612-4ec5-a9a1-007f4ea2dc65" />


## Objectives
- Analyze hotel revenue performance
- Track occupancy and capacity utilization
- Monitor booking and cancellation trends
- Identify high-performing room categories
- Support business decision-making using data visualization

## Tools & Technologies
- Power BI
- Microsoft Excel
- Data Cleaning
- DAX
- Data Visualization

## Dashboard Insights

### Financial Overview
- Total Revenue
- Revenue Growth (MoM & WoW)
- Average Daily Rate (ADR)
- Revenue per Available Room (RevPAR)

### Occupancy Analysis
- Occupancy Rate by Hotel
- Seasonal Occupancy Trends
- Occupancy Growth Analysis

### Booking Insights
- Booking Lead Time
- Average Length of Stay (ALOS)
- Room Category Performance

### Cancellation Analysis
- Cancellation Rate
- Lost Revenue due to Cancellations
- Cancellation Trend Analysis

## Key Features
- Interactive Power BI Dashboard
- Dynamic Filters and Slicers
- KPI Tracking
- Business Performance Analysis
- Clean and Professional Visualizations
